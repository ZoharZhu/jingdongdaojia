# Project
“京东到家”移动端项目，使用 vue3 + vuex + vue-router + vue-cli + scss 等技术，主要是用于学习 vue3 全家桶相关技术栈。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
